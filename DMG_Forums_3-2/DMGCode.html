Links:
    [url]http://www.google.com[/url] Becomes http://www.google.com

    [url=http://www.google.com]Google[/url] Becomes Google

Multimedia:
    [img]images/banner.gif[/img] Displays the image on the screen.

    [img=left]images/banner.gif[/img] Displays the image left-aligned.

    [img=right]images/banner.gif[/img] Displays the image right-aligned.

    [img=center]images/banner.gif[/img] Displays the image centered.

    [MemberPhoto=#] Displays the member photo with the specified ID.

    [flash]flashmovie.swf[/flash] Displays a flash movie.

    [YouTube]YOUTUBE_URL[/YouTube] Displays a YouTube movie.

    [YouTubeLite]YOUTUBE_URL[/YouTubeLite] Displays a YouTube movie in low quality.

    [YouTubeHD]YOUTUBE_URL[/YouTubeHD] Displays a YouTube movie in HD (if available).

Text Formatting:
    [b]TEXT[/b] Becomes Text.

    [i]TEXT[/i] Becomes TEXT.

    [u]TEXT[/u] Becomes TEXT.

    [s]TEXT[/s] Becomes TEXT.

    [highlight]TEXT[/highlight] Becomes TEXT.

    [highlight=COLOR_NAME]TEXT[/highlight] Outputs highlighted text with the given color.

    [font=FONT_NAME]TEXT[/font] Changes the font face to the entered font name.

    [color=COLOR_NAME]TEXT[/color] Changes the font color to the entered value.

    [size=#]TEXT[/size] Changes the font size to the entered number.

    [left]TEXT[/left] Aligns text left.

    [right]TEXT[/right] Aligns text right.

    [center]TEXT[/center] Centers the text.

    X[sup]2[/sup] Becomes X2.

    O[sub]2[/sub] Becomes O2.

Other:
    [quote]TEXT[/quote] Creates a quote box.

    [code]TEXT[/code] Formats text for posting code.

    [pre]TEXT[/pre] Outputs content as pre-formatted text.

    [hr] Outputs a horizontal bar.
