Full HTML code is allowed on all content pages. The following codes can be used to display data from the database or for special formatting. Regular DMG Forum Code also applies on custom content pages.

Special Functions:
[Menu=H]Displays the main menu horizontally.
[Menu=V]Displays the main menu vertically.
[ForumMenu=H/V]Displays the forums menu horizontally or vertically.
[PageMenu Align=H/V]Displays the content page menu horizontally or vertically.
[PageMenu Align=H/V Parent=#]Optional "Parent" argument to show sub-pages from a specified parent page.
[LatestTopics Topics=NUM]TITLE[/LatestTopics]Outputs the latest (NUM) forum topics in a content box with the title (TITLE).
[LatestTopicsNoBox Topics=NUM]Outputs the latest (NUM) forum topics without a content box.
[LatestTopics Topics=NUM ForumID=FID]TITLE[/LatestTopics]Outputs the latest (NUM) forum topics from a specific forum (FID).
[LatestBlogs=NUM]TITLE[/LatestBlogs]Outputs the latest (NUM) user blogs in a content box with the title (TITLE).
[LatestBlogsNoBox=NUM]Outputs the latest (NUM) user blogs without a content box.
[FeaturedTopics Days=# Max=# Chars=#]TITLE[/FeaturedTopics]Shows a formatted box with featured topics. (Days) is the number of days topics will stay featured, (Max) is the maximum number of featured topics to show, (Chars) is the amount of characters to show of the topic's content, TITLE is the name of the box. All variables are optional.
[PhotoGallery ID=NUM1 Columns=NUM2]Displays the photo gallery ID = (NUM1), with (NUM2) columns.
[ImageRotator=NUM]Displays the image rotator with ID = (NUM).
[ContentBox=BOX_TITLE]BOX_TEXT[/ContentBox]Creates a formatted box around a piece of text along with a title.
[ContentBox]BOX_TEXT[/ContentBox]Creates a formatted box around a piece of text with no title.
Database Information:
[PageTitle]Displays the page title.
[ForumLogo]Displays the forum logo URL.
[HomeUrl]Displays the home page URL.
[FontFace]Default font face.
[FontColor]Default font color.
[FontSize]Default font size.
[HeaderColor]Header background color.
[HeaderFontColor]Header font color.
[HeaderFontSize]Header font size.
[FooterColor]Footer background color.
[FooterFontColor]Footer font color.
[FooterFontSize]Footer font size.
[SubHeaderColor]Sub-header background color.
[SubHeaderFontColor]Sub-header font color.
[BorderColor]Default table border color.
[ButtonColor]Form buttons color.
[LinkColor]Link color.
[LinkDecoration]Link decoration.
[VLinkColor]Visited link color.
[VLinkDecoration]Visited link decoration.
[ALinkColor]Active link color.
[ALinkDecoration]Active link decoration.
[HLinkColor]Hover link color.
[HLinkDecoration]Hover link decoration.
[BackgroundColor]Page background color.
[ScrollbarColor]Page scrollbar color.
[TopicsFontSize]Font size for topics.
[TopicsFontColor]Font color for topics.
[TableBGColor2]Alternation table BG color 2.
[LoginFontColor]Login box font color.
Custom Links:
[RegisterLink]Displays a Register button for non-logged in users and an Edit Profile button for logged in users.
[AdminLink]Displays an Administration link when an admin is logged in.
[PMLink]Displays a link to Private Messaging for logged in users.
[Page=PAGE_ID]LINK_TITLE[/Page]Creates a link to a given page.
[Forums]LINK_TITLE[/Forums]Creates a link to the forums page.
[Forum=FORUM_ID]LINK_TITLE[/Forum]Creates a link to a given forum.
[CreateTopicLink]LINK_TITLE[/CreateTopicLink]Creates a link to the "Post New Topic" page.
